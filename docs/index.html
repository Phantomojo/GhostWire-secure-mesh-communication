Features

  • End-to-end encrypted mesh
  • Multi-transport: Bluetooth, WiFi, LoRa, WebRTC
  • Protocol adapters: Briar, Meshtastic, Matrix
  • Sybil defense, quotas, blacklists
  • Disaster triggers, reputation, federation
